Evaluation methodology:

Overall:
	If the top 1 predicted label is in the labels list for the example then TP++ otherwise FP++

Per label (multi-label prediction for N labels in a test sample):
	TP++ for all labels in the top N predicted labels that are also in the test example labels
	FP++ for all labels in the top N predicted labels that are not in the test example labels
	TN++ for all labls that are not in top N predicted labels and are not in the test example labels
	FN++ for all labls that are not in top N predicted labels but are in the test example labels

# Usage
<!-- usage -->
```sh-session
$ npm install -g @microsoft/bf-cli-config
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
@microsoft/bf-cli-config/0.0.0 darwin-x64 node-v12.1.0
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
